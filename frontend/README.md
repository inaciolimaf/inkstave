# Inkstave frontend

The Inkstave web client: a **Vite + React + TypeScript** app (with Tailwind,
shadcn/ui, CodeMirror 6 and PDF.js). At spec 01 this is only a placeholder —
the actual Vite/React application is scaffolded in **spec 09**.

Managed with [`pnpm`](https://pnpm.io/) via the root `pnpm-workspace.yaml`.
